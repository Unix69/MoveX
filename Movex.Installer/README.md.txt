Microsoft Visual Studio 2017 Projects Installer
===============================================

E' un tool che permette di creare pacchetti di installazione per progetti realizzati con Microsoft Visual Studio 2017. L'installazione di questo tool aggiuntivo (non disponibile by default) viene garantita tramite il marketplace interno di Microsoft Visual Studio 2017.

PROCEDURA
=========

1. Dal menù di navigazione seguire: Tools > Extensions and Updates...
2. Seguire Online > Visual Studio Marketplace
3. Nella barra di ricerca digitare la query di ricerca "Installer"
4. Selezionare "Microsoft Visual Studio 2017 Installer Projects" per il download
5. L'installazione avviene con il riavvio di Visual Studio.
